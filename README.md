## `$ ls -la ./selected-work`

### [`gitzero/`](https://github.com/Ivansost/gitzero)

A Python CLI for scanning repositories for signals consistent with AI-assisted code. It combines 25+ git-history and static-analysis signals, explainable scoring, false-positive guards, JSON/CSV exports, and an optional calibrated Random Forest model. [Read the case study →](https://www.ivansostaric.com/projects/gitzero)

### [`aml-network-analytics/`](https://www.ivansostaric.com/projects/aml)

An anti-money laundering analysis that turns financial transactions into a network. SQL, Python, NetworkX, risk scoring, and Power BI work together to surface suspicious clusters, central accounts, and fund-dispersion patterns. [Read the case study →](https://www.ivansostaric.com/projects/aml)

### [`credit-risk-scoring/`](https://github.com/Ivansost/Credit-Risk-Scoring)

A machine-learning study of 30,000 customer records using Pandas, SQL, and Random Forest. The model reached **0.86 ROC-AUC**, with Power BI used to communicate default-risk patterns across customer cohorts. [Read the case study →](https://www.ivansostaric.com/projects/credit-risk)

### [`travelpal/`](https://github.com/Ivansost/gdschacks25)

A full-stack, AI-powered sustainable trip planner built at GDSC Hacks 2025 with Next.js, Convex, Clerk, Gemini, and Vapi. It creates personalized itineraries while making each trip's environmental impact easier to understand. [Read the case study →](https://www.ivansostaric.com/projects/travelpal)
